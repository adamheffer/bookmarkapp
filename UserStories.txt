User Stories for Bookmark Manager


As a user, I’d like to be able to see a list of bookmarks so that I can visit the saved addresses.


As a user, I’d like to be able to add new bookmarks so that I can visit them at a later date.


As a user, I’d like to be able to delete bookmarks so that when they’re no longer required I can remove them.


As a user, I’d like to be able to update bookmarks when addresses or descriptions change.


As a user, I’d like to be able to comment on bookmarks so that I know what sites they’re for.


As a user, I’d like to be able to tag bookmarks so that I can organise them.


As a user, I’d like to be able to filter bookmarks by tag so that I can search them more easily.


As a user, I’d like to be able to only manage my own bookmarks so that no one else can edit them.
